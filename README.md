# 🚀 AI Career Copilot

An intelligent Resume Analyzer and Career Guidance Web App built using Python and Streamlit.

---

## 🔥 Features

* 📄 Resume Upload (PDF)
* 🧠 Skill Extraction
* 🎯 Job Role Matching
* 📊 ATS Resume Score
* 🚀 Skill Gap Analysis
* 📈 Graph Visualization

---

## 🛠️ Tech Stack

* Python
* Streamlit
* PyPDF2
* Matplotlib

---

## ▶️ How to Run

```bash
pip install streamlit PyPDF2 matplotlib
streamlit run app.py
```
